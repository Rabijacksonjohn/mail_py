Mail_py:
Mail_py is a lightweight Python tool that monitors your inbox for critical emails (like Interviews, Meetings, or Internships) so you never miss an important career opportunity while traveling or busy.

The Problem:
When applying for jobs and internships, you often have to wait weeks for a reply. If an important invitation arrives while you are traveling, busy, or offline, it is incredibly easy for it to get buried under a mountain of unwanted spam and newsletters.

The Solution:
Instead of scrolling through your entire inbox, Mail_py acts as an automated filter. It scans your received mail for high-priority keywords and immediately surfaces the exact emails that require your urgent attention.

Setup
Install dependencies:

Bash:
pip install python-dotenv
Create a .env file in the same folder as your script:

Code snippet
EMAIL="your_email@gmail.com"
PASSWORD="your_gmail_app_password"
Run the program:

Bash:
python main.py
